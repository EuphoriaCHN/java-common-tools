|:----|:----|:----|:----|

|isEmpty|Object[] objects|boolean|检测 objects 是否为空|
|equals|String str1, String str2|boolean|检验两字符串是否相等|
|equals|Integer n1, Integer n2|boolean|判断 n1 和 n2 是否相等|
